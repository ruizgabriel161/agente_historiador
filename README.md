# LLM Agent Starter

Starter project para criação de **agentes LLM** usando **LangChain** e **LangGraph**, com uma arquitetura base reutilizável, focada em boas práticas, padronização e produtividade.

Este repositório foi pensado para servir como **template** para novos projetos, evitando retrabalho na configuração inicial de ambiente, dependências e estrutura.

---

## 🎯 Objetivo

Fornecer uma base sólida para desenvolvimento de aplicações com LLM, incluindo:

- Orquestração de agentes com LangGraph
- Chains, tools e memória com LangChain
- Estrutura organizada e extensível
- Padronização de código e tipagem
- Ambiente reproduzível com `uv`

---

## 🧱 Stack Tecnológica

- **Python** 3.11+
- **LangChain**
- **LangGraph**
- **uv** (gerenciador de dependências e ambiente)
- **Ruff** (lint + format)
- **Pyright** (type checking)
